京盛传媒智能体 企业版 (FULL)
-------------------------------------
功能概述：
- 多源市场数据采集（亚马逊、Etsy、TikTok、YouTube、政策中心等）
- 智能分析（AI推荐、异常检测、ROI计算、竞品矩阵）
- 自动日报邮件
- UI 界面（浅色模式）+ 数据来源溯源
- API在线管理

使用步骤：
1. 安装 Python 3.10 或更高版本
2. 打开 PowerShell，进入本目录：
   cd D:\京盛传媒智能体_企业版
3. 创建虚拟环境：
   python -m venv .venv
4. 激活虚拟环境：
   .\.venv\Scripts\activate
5. 安装依赖：
   pip install --upgrade pip
   pip install -r requirements.txt
6. 启动：
   双击 smart_start.bat
7. 浏览器会自动打开 http://localhost:8501
